# Agencia_de_Viajes_On_tour

## Prototipo - Agencia de Viajes On Tour

Este repositorio contiene el prototipo de la aplicación web **Agencia de Viajes On Tour**, desarrollado en Figma.

### Pantallas implementadas

- Inicio de sesión
- Dashboard del apoderado
- Historial de depósitos
- Descarga de documentos (contrato y póliza)
- Panel de administración

### Funcionalidades representadas

- Consulta de saldo acumulado.
- Visualización del avance de pagos.
- Revisión del historial de depósitos.
- Descarga de contrato en PDF.
- Descarga de póliza en PDF.
- Acceso al panel de administración.
